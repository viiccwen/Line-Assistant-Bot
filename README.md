# LineBot Line多功能機器人


## Introduction - 介紹

為了使我們的生活更加方便，我決定開發一個多功能機器人，類似於個人助理，能夠在某些時候節省我們許多時間去翻閱、處理資訊。

## Function - 功能

* 美食推薦
* 景點推薦
* 天氣預覽
* AI對話
